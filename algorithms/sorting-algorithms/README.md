# Sorting Algorithms
1. Slow Sorting Algorithms:
 - Insertion Sort - `O(n^2)`
 - Selection Sort - `O(n^2)`
 - Bubble Sort - `O(n^2)`
2. Fast Sorting Algorithms:
 - Heap Sort - `O(n*log n)`
 - Merge Sort - `O(n*log n)`
 - Quick Sort - `O(n*log n)`
 - Counting Sort - `O(n+k)`
 - Radix Sort - `O(n + k)`
